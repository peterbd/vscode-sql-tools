# Change Log

All notable changes to the "px-sql-tools" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.0] - 2025-11-05

- First public preview of PX SQL Tools (formerly SQL Toolbelt Lite).
- Added schema-aware completions, hovers, and JOIN hints driven by MSSQL metadata.
- Implemented wildcard expansion, snippet pack, and custom snippet loader.
- Introduced SQL history tree view with search, favorites, and recovery support.
- Added script-as-ALTER/CREATE commands backed by OBJECT_DEFINITION.
